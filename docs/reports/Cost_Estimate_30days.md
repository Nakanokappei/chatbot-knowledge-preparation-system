## コスト削減オプション

| 手段 | 削減効果 | 適用シナリオ |
|------|---------|------------|
| **RDS Reserved Instance (1年)** | 常時起動の 30〜40% 削減 | 本番確定後 |
| **Fargate Spot** | 最大 70% 削減 | 全シナリオ（既に想定に含む） |
| **Bedrock Batch Inference** | 50% 削減 | 本番の定期バッチ |
| **Aurora Serverless v2** | アイドル時 90% 削減 | 開発環境のDB |
| **RDS 夜間停止** | 開発環境の 60% 削減 | シナリオ B |
| **Savings Plans (Fargate)** | 常時起動の 20% 削減 | 本番 |
